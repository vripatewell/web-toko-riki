Panduan Penggunaan

Langkah 1: Isi Nama Subdomain

Ketik nama subdomain yang Anda inginkan. Gunakan huruf kecil tanpa spasi (contoh: tokosaya).

Langkah 2: Pilih Domain Utama

Pilih salah satu domain utama dari daftar yang tersedia. Ini akan menjadi akhiran dari subdomain Anda (contoh: tokosaya.bisnis-panel.web.id).

Langkah 3: Pilih Tipe Record

(A) untuk IP Address: Gunakan ini jika Anda ingin mengarahkan subdomain ke IP VPS atau Panel.
(CNAME) untuk Target: Gunakan ini jika Anda ingin mengarahkan ke layanan seperti Vercel, Netlify, dll.

Langkah 4: Isi Target

Masukkan IP Address atau target CNAME sesuai dengan Tipe Record yang Anda pilih pada langkah sebelumnya.

Langkah 5: Atur Proxy

Aktifkan Status Proxy (awan oranye) untuk menyembunyikan IP asli Anda dan mendapatkan perlindungan dari Cloudflare. tetapi untuk panel atau vercel lebihh baik di matikan karena agar tidak terjadi nya eror. soalnya kalo mau nyalain proxy harus di set dari dalam doamik nya jadi aga rumit.
